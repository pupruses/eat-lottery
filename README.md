# 🍱 吃饭盲盒 (Eat Lottery)

一个帮你决定"今天吃什么"的智能辅助决策系统，由 Trae 协助构建。

## 核心功能

1.  **菜品抽卡**：随机生成几天的午餐和晚餐。
2.  **动态配额**：根据就餐人数和健康目标（蔬菜/肉类克数），自动计算所需食材重量。
3.  **智能采购单**：自动合并相同食材，生成最终购物清单。
4.  **营养雷达**：检测每日菜单是否缺乏肉类或蔬菜，给出"红灯"预警。
5.  **自定义菜谱**：随时录入新菜品，定义食材构成（素/肉/碳水）。

## 快速开始

本项目是一个标准的 Vite + React + TypeScript 项目。

### 1. 安装依赖

确保你已经安装了 [Node.js](https://nodejs.org/) (推荐 v18+)。

```bash
npm install
# 或者
yarn
# 或者
pnpm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

打开浏览器访问控制台输出的地址 (通常是 http://localhost:5173)。

### 3. 构建生产版本

```bash
npm run build
```

## 架构说明

*   **src/lib/logic.ts**: 核心算法模块，包含配额计算 (`calculateDailyIngredients`) 和 采购单聚合 (`generateShoppingList`)。
*   **src/components/**: UI 组件库。
    *   `Sidebar`: 全局控制台 (人数、天数、目标设置)。
    *   `DishManager`: 菜品录入引擎。
    *   `DayCard`: 单日菜单展示与换菜逻辑。
    *   `ShoppingList`: 最终交付清单。
*   **src/App.tsx**: 应用主入口，负责状态管理和数据持久化 (LocalStorage)。

## 使用指南

1.  在左侧 **全局控制台** 设置吃饭人数和天数。
2.  调整 **健康目标** (例如每人每天 400g 蔬菜)。
3.  查看右侧生成的 **本周食谱**。
4.  如果不满意某顿饭，点击卡片右上角的 **刷新图标** 换菜 (支持指定换肉/换素)。
5.  在底部查看自动生成的 **智能采购单**，直接照着买菜即可！
